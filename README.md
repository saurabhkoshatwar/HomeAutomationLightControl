# Intelligent-Home-Assistant

## Steps to RUN
1. Run Flask server on RaspberryPI <br />
  `python main.py` 
1. Initiate port forwarding using ngrok on RaspberryPI  <br />
  ` ./ngrok http 8000 `
1. Configure this address obtained on ngrok terminal into diaglogflow
